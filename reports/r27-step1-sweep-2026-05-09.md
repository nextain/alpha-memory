## 핵심 발견 — Plateau 확인

**모든 threshold 값에서 noise band ±2pp 안**. 즉 minConfidence 의 visible
효과 X. reviewer 1 가설 (3) 의 정확한 검증:

> R2.5 v2 chain + 보존 우선 통합 효과가 dominate. mode='latest' 가
> status='active' 만 노출 → MMR / threshold 등 ranking 강화 mechanism 이
> *이미 chain 이 attribute 별 1개로 압축한 set* 위에서 작동 → redundant.

## 해석

1. **minConfidence axis ceiling 도달** — naia 의 base retrieval (cosine +
   BM25 + RRF + KG + chain) 이 이미 strict ranking. threshold 추가 효과 X.
2. **mem0 \"97.8% junk\" 회피 가설은 naia 에서 invalid** — naia 가 *이미
   confidence 충분* (cosine 0.7 recall@20 = 76.8%).
3. **#27 Step 1 자체 의미 limited** — noise band ±2pp 안에서는 threshold
   가 정확도 향상 X. 그러나 *abstention* (#9) 의 *signal* 로는 가치 (결과 0
   인 경우 caller 가 abstain 결정 가능).
