# Benchmark Reports / 벤치마크 보고서

Benchmark results comparing Alpha Memory against other memory systems.
Alpha Memory와 다른 메모리 시스템의 비교 벤치마크 결과입니다.

## Structure / 구조

```
reports/
├── README.md         # This file
├── r1-ko-*.md        # Round 1 — Korean fact bank (1000 facts)
└── r2-en-*.md        # Round 2 — English fact bank (1000 facts)
```

Reports are generated by `src/benchmark/comparison/run-comparison.ts`.
보고서는 `src/benchmark/comparison/run-comparison.ts`로 생성됩니다.
